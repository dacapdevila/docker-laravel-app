.env

Creando el Dockerfile del servicio

  • Crear nuevo branch: " base-1-local ".

Para el frontend-service

  • En el caso de que dentro de frontend-service/ no exista un archivo .env, se debe crear:

    cp .env.example .env

  • Editar el archivo .env

    APP_NAME="Frontend Service" APP_ENV=local APP_KEY= APP_DEBUG=true APP_URL=http://localhost:8000 DB_CONNECTION=null #DB_HOST=db #DB_PORT=3306 #DB_DATABASE=frontend #DB_USERNAME=frontend_user #DB_PASSWORD=secret_password # These variables will be used on docker-compose.yml ...
