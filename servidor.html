Docker Tutorial Help

Servidor

  • Ingresar al droplet:

    ssh root@YOUR_DROPLET_IP

  • Instalar Git:

    apt update apt install git -y

  • Instalar Docker:

    apt install apt-transport-https ca-certificates curl software-properties-common -y curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add - add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" apt update apt install docker-ce -y systemctl start docker systemctl enable docker docker --version

  • Generar SSH en el droplet:

    ssh-keygen -t ed25519 -C "xxx@yyy.com" cat ~/.ssh/id_ed25519.pub
