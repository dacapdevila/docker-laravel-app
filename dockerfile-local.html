Dockerfile (local)

Para el frontend-service

  • Crear Dockerfile:

    touch Dockerfile

  • Copiar el siguiente contenido al Dockerfile:

    # Placed in frontend-service/Dockerfile FROM php:8.2-fpm # Arguments defined in docker-compose.yml ARG user ARG uid # Install system dependecies RUN apt-get update && apt-get install -y \ git \ curl \ libpng-dev \ libonig-dev \ libxml2-dev \ zip \ unzip # Clear cache RUN apt-get clean && rm -rf /var/lib/apt/lists/* # Install PHP extensions RUN docker-php-ext-install pdo_mysql mbstring exif pcntl bcmath gd # Get latest Composer COPY --from=composer:latest /usr/bin/composer /usr/bin/composer # Create system user to run cComposer and Artisan Commands RUN useradd -G www-data,root -u $uid -d /home/$user $user RUN mkdir -p /home/$user/.composer && \ chown -R $user:$user /home/$user # Set working directory WORKDIR /var/www/frontend USER $user
