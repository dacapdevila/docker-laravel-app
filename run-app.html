Docker Tutorial Help

Run App

Corriendo la aplicación

  • En el caso de necesitar eliminar lo construido:

    docker-compose down

  • Para crear la aplicación es necesario correr comandos de Docker:

    docker-compose build frontend-service docker-compose build kitchen-service docker-compose build order-service docker-compose build warehouse-service

  • Una vez que termine de crear el servicio, es necesario levantar el servicio con:

    docker-compose up -d

  • Para ver los servicios Docker que esten funcionando:

    docker-compose ps

  • Aún es necesario correr algunos comandos más:

    docker-compose exec frontend-service ls -l docker-compose exec kitchen-service ls -l docker-compose exec order-service ls -l docker-compose exec warehouse-service ls -l

  • Correr compose install en frontend-service:

    docker-compose exec frontend-service rm -rf vendor composer.lock docker-compose exec frontend-service composer install

  • Correr compose install en kitchen-service:

    docker-compose exec kitchen-service rm -rf vendor composer.lock docker-compose exec kitchen-service composer install

  • Correr compose install en order-service:

    docker-compose exec order-service rm -rf vendor composer.lock docker-compose exec order-service composer install

  • Correr compose install en warehouse-service:

    docker-compose exec warehouse-service rm -rf vendor composer.lock docker-compose exec warehouse-service composer install

  • Crear el APP_KEY en frontend-service:

    docker-compose exec frontend-service php artisan key:generate

  • Crear el APP_KEY en kitchen-service:

    docker-compose exec kitchen-service php artisan key:generate

  • Crear el APP_KEY en order-service:

    docker-compose exec order-service php artisan key:generate

  • Crear el APP_KEY en warehouse-service:

    docker-compose exec warehouse-service php artisan key:generate

  • Correr migraciones en kitchen-service:

    docker-compose exec kitchen-service php artisan migrate

  • Correr migraciones en order-service:

    docker-compose exec order-service php artisan migrate

  • Correr migraciones en warehouse-service:

    docker-compose exec warehouse-service php artisan migrate
