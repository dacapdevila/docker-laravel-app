Nginx

Seteando la configuración de Nginx y los database dump files

  • Crear una carpeta en docker-compose/nginx para ubicar toda la configuración de nginx:

    mkdir -p docker-compose/nginx

  • Crear el archivo nginx.conf dentro de ese directorio:

    touch docker-compose/nginx/nginx.conf

  • Copiar el siguiente contenido en nginx.conf:

    server { listen 80; listen [::]:80; server_name dariocapdevila.com.ar www.dariocapdevila.com.ar; return 301 https://$host$request_uri; } server { listen 443 ssl; listen [::]:443 ssl; server_name dariocapdevila.com.ar www.dariocapdevila.com.ar; root /var/www/frontend/public; ssl_certificate /etc/letsencrypt/live/dariocapdevila.com.ar/fullchain.pem; ssl_certificate_key /etc/letsencrypt/live/dariocapdevila.com.ar/privkey.pem; ssl_protocols TLSv1.2 TLSv1.3; ssl_prefer_server_ciphers on; ssl_ciphers HIGH:!aNULL:!MD5; add_header X-Frame-Options "SAMEORIGIN"; add_header X-Content-Type-Options "nosniff"; index index.php; charset utf-8; location / { try_files $uri $uri/ /index.php?$query_string; } location /kitchen { alias /var/www/kitchen/public; try_files $uri $uri/ @kitchen; location ~ \.php$ { fastcgi_pass kitchen-service:9000; fastcgi_param SCRIPT_FILENAME $request_filename; include fastcgi_params; } } location @kitchen { rewrite /kitchen/(.*)$ /kitchen/index.php?/$1 last; } location /order { alias /var/www/order/public; try_files $uri $uri/ @order; location ~ \.php$ { fastcgi_pass order-service:9000; fastcgi_param SCRIPT_FILENAME $request_filename; include fastcgi_params; } } location @order { rewrite /order/(.*)$ /order/index.php?/$1 last; } location /warehouse { alias /var/www/warehouse/public; try_files $uri $uri/ @warehouse; location ~ \.php$ { fastcgi_pass warehouse-service:9000; fastcgi_param SCRIPT_FILENAME $request_filename; include fastcgi_params; } } location @warehouse { rewrite /warehouse/(.*)$ /warehouse/index.php?/$1 last; } location = /favicon.ico { access_log off; log_not_found off; } location = /robots.txt { access_log off; log_not_found off; } error_page 404 /index.php; location ~ \.php$ { fastcgi_pass frontend-service:9000; fastcgi_param SCRIPT_FILENAME $realpath_root$fastcgi_script_name; include fastcgi_params; } location ~ /\.(?!well-known).* { deny all; } }

  • Crear el archivo nginx.local.conf dentro de ese directorio:

    touch docker-compose/nginx/nginx.local.conf

  • Copiar el siguiente contenido en nginx.conf:

    server { listen 80; listen [::]:80; server_name localhost; root /var/www/frontend/public; add_header X-Frame-Options "SAMEORIGIN"; add_header X-Content-Type-Options "nosniff"; index index.php; charset utf-8; location / { try_files $uri $uri/ /index.php?$query_string; } location /kitchen { alias /var/www/kitchen/public; try_files $uri $uri/ @kitchen; location ~ \.php$ { fastcgi_pass kitchen-service:9000; fastcgi_param SCRIPT_FILENAME $request_filename; include fastcgi_params; } } location @kitchen { rewrite /kitchen/(.*)$ /kitchen/index.php?/$1 last; } location /order { alias /var/www/order/public; try_files $uri $uri/ @order; location ~ \.php$ { fastcgi_pass order-service:9000; fastcgi_param SCRIPT_FILENAME $request_filename; include fastcgi_params; } } location @order { rewrite /order/(.*)$ /order/index.php?/$1 last; } location /warehouse { alias /var/www/warehouse/public; try_files $uri $uri/ @warehouse; location ~ \.php$ { fastcgi_pass warehouse-service:9000; fastcgi_param SCRIPT_FILENAME $request_filename; include fastcgi_params; } } location @warehouse { rewrite /warehouse/(.*)$ /warehouse/index.php?/$1 last; } location = /favicon.ico { access_log off; log_not_found off; } location = /robots.txt { access_log off; log_not_found off; } error_page 404 /index.php; location ~ \.php$ { fastcgi_pass frontend-service:9000; fastcgi_param SCRIPT_FILENAME $realpath_root$fastcgi_script_name; include fastcgi_params; } location ~ /\.(?!well-known).* { deny all; } }
